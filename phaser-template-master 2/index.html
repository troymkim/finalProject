MB Phaser Templates

The templates and examples below are meant to help you with your final project. Clone this repo and navigate to the desired folder to find the sample code.

Remember: to run a local server, open Terminal, navigate to the folder using cd, and type: python -m SimpleHTTPServer

Don't see the template that you need? Request a tutorial

Minimal Template

This template contains the minimum requirements for a Phaser game with an animated sprite.

Ninja Physics

Template for using Phaser's Ninja Physics.

Platformer

Basic elements of a platformer game.

Ship

A starting point for top-down games such as Asteroids or New Rally X.

States

Learn how to create game states and check out how the x,y coordinates work in Phaser.

Simple Game

This template uses Arcade Physics and has examples of how to handle states, console logging and alerts, scoring, displaying text, manipulating sprites, collision detection, and designing basic worlds.
